# 📄 PDF para PNG

Script Python simples e eficiente para converter páginas de arquivos PDF em imagens PNG de alta resolução (300 DPI).

## 🚀 Como Usar

1. **Instale as dependências:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure o arquivo:**
   Coloque seu PDF na pasta do projeto e verifique se o nome do arquivo no `converter.py` está correto.

3. **Execute o script:**
   ```bash
   python converter.py
   ```

As imagens serão geradas automaticamente na pasta raiz (ex: `pagina_1.png`).

## 🛠️ Tecnologias

- **PyMuPDF**: Para leitura e renderização do PDF.
- **Pillow**: Para processamento e salvamento das imagens.
